# Análise de Desempenho de Alunos

Projeto desenvolvido em Python utilizando a biblioteca **pandas** para análise de dados de alunos a partir de um arquivo CSV.

O sistema funciona via terminal e permite ao usuário interagir com um menu para visualizar e gerar diferentes análises.

## Funcionalidades

* Visualizar tabela completa dos alunos
* Calcular média geral das notas
* Analisar média das notas por idade
* Identificar maior e menor nota por idade
* Contar quantidade de alunos por idade
* Contar alunos com nota maior que 7 por idade
* Listar alunos com nota baixa
* Identificar o aluno destaque (maior nota)
* Gerar um relatório completo com todas as análises


## Tecnologias Utilizadas

* Python 3
* pandas
